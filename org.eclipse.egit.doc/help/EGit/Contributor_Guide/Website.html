Website

The EGit and JGit websites are located in Git repositories which are configured for Gerrit code review.

EGit

  • File > Import > Git > Projects from Git
  • Select URL
    • HTTPS protocol: https://eclipse.gerrithub.io/eclipse-egit/egit-website
    • SSH protocol: ssh://user@eclipse.gerrithub.io:29418/eclipse-egit/egit-website
  • in Repositories View on node "origin" click "Gerrit Configuration..." and select branch "master", then changes you push to upstream will end up in Gerrit for review and can be submitted there

JGit

  • File > Import > Git > Projects from Git
  • Select URL
    • HTTP protocol: https://eclipse.gerrithub.io/eclipse-jgit/jgit-website
    • SSH protocol: ssh://user@eclipse.gerrithub.io:29418/eclipse-jgit/jgit-website
  • in Repositories View on node "origin" click "Gerrit Configuration..." and select branch "master", then changes you push to upstream will end up in Gerrit for review and can be submitted there
